# DevSecMLOps - Project red wire

This threaded project allows us to follow the implementation of a DevSecMLOps project with AWS and MLFlow for @ESGI.


This project uses the following AWS components:
- SageMaker
- S3
- EC2
- API Gateway
- Lambda
- ELB
- ECS
- RDS
